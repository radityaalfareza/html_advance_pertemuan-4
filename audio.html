Audio


Kode ini menampilkan dua elemen audio pada halaman web menggunakan tag audio. Setiap elemen memiliki kontrol pemutar (play, pause, volume) sehingga pengguna dapat memutar file audio yang disediakan. File audio yang digunakan berada dalam folder /gambar/audio/ dengan nama "suara.mp3" dan "suara1.mp3". Hal ini memungkinkan pengguna untuk mendengarkan suara langsung dari halaman web tanpa perlu mengunduhnya terlebih dahulu.ketika diberi autoplay maka audio otomatis terputar dengan suara. namun jika diberi muted maka audio terputar tanpa suara
